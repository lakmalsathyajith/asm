## Setup Guide
- Clone
- Compoeser install
- Add database details
  >DB_CONNECTION=mysql
  >DB_HOST=127.0.0.1
  >DB_PORT=3306
  >DB_DATABASE
  >DB_USERNAME
  >DB_PASSWORD
- Generate app key
- PHP artisan migrate
- NPM install (node 9.8.0)
- NPM run dev or NPM run watch to build
- Map with server url
